# TOEIC Vocab Steps

スマートフォンのホーム画面に追加して使える、TOEIC語彙学習用の静的PWAです。

## 収録内容

- 200点から900点まで、100点刻みの8レベル
- 全4,686項目
- 3,418単語、1,268熟語・定型表現
- 各レベルの収録数に合わせて、100項目ごとのグループを動的に選択
- 4択テスト、単語帳、検索、レベル絞り込み、チェック単語、誤答リスト
- 選択肢を隠すモード
- 端末の英語音声による読み上げ

音声はブラウザの Web Speech API を使います。個別の音声ファイルは不要で、追加単語・追加熟語を含む全項目を読み上げ対象にしています。

## 起動方法

このフォルダを静的Webサーバーで公開します。ローカルで試す場合は、このフォルダ内で次を実行します。

```powershell
python -m http.server 4173
```

ブラウザで `http://localhost:4173` を開きます。PWAとして公開する場合は、HTTPS対応の静的ホスティングに配置してください。

## ファイル構成

- `index.html`: アプリ本体HTML
- `styles.css`: スタイル
- `app.js`: クイズ、単語帳、学習状態、音声読み上げのロジック
- `vocab-data.js`: TOEIC語彙データ
- `sw.js`: オフラインキャッシュ用サービスワーカー
- `manifest.webmanifest`: PWAマニフェスト
- `icon.svg`: アプリアイコン
- `VOCAB_LICENSE.md`: 語彙データのライセンス説明
- `THIRD_PARTY_LICENSE-CC-BY-SA-4.0.txt`: 第三者データのライセンス

## 語彙データについて

公開されている教育用語彙リストを用途別に組み合わせ、英語の一般頻度、TOEICで出やすい案内・業務連絡・手続き・契約文書の語彙、熟語・定型表現を加えて構成しています。

点数帯は学習上の目安です。公式のスコア別単語表ではなく、習得によりスコアを保証するものでもありません。
